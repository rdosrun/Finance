Investment & Project Evaluation

Concept Formula Description
Net Present Value (NPV) NPV = ∑ Ct / (1 + r)t − C0 Present value of cash inflows minus initial investment
Internal Rate of Return (IRR) IRR is the rate r that makes NPV = 0 Usually found using Excel, financial calculator, or software
Profitability Index (PI) PI = ∑ Ct / (1 + r)t / C0 Value created per unit of investment
Payback Period Years before recovery + Unrecovered amount / Next year’s cash flow Time to recover initial investment
Discounted Payback Period Similar to payback, but using discounted cash flows Accounts for time value of money
