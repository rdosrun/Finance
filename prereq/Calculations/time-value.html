Time Value of Money Formulas

Concept Formula Description
Future Value (FV) FV = PV × (1 + r)t Value of investment after time t
Present Value (PV) PV = FV / (1 + r)t Today's value of a future amount
Rate (r) r = (FV / PV)1/t − 1 Interest rate or return per period
Number of Periods (t) t = log(FV / PV) / log(1 + r) Time required to reach a future value
